# Project Goal

A personal Telegram AI assistant for a single user, running locally on a Mac.

## Current capabilities

- **Finance tracking** — log income/expenses in natural language, track budgets and balances, auto-save 10% of income to a piggy bank, sync everything to Google Sheets
- **Web search** — retrieve current information via Tavily with cross-source validation
- **Currency rates** — ExchangeRate.host + National Bank of Georgia official rates
- **Voice messages** — transcribe to text via Groq Whisper, then process as normal message
- **Obsidian notes** — create Markdown notes in the local vault via the desktop agent
- **General AI chat** — conversation with history via Groq (llama-3.3-70b-versatile)

## Current priority

Stabilizing the finance tracking system. Finance functionality should be reliable before expanding other capabilities.
